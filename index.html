HPC + machine learning systems

Making machine learning models run fast on real hardware.

I am a Staff Software Engineer at Blaize, where I work on optimizing deep learning models for the Blaize Graph Streaming Processor (GSP) architecture. Blaize GSP is a graph-native computing architecture designed from the ground up for efficient artificial intelligence applications at the edge.

My work sits at the intersection of high performance computing and machine learning. I am particularly interested in parallel, scalable ML algorithms and systems that accelerate these models across hardware architectures.

Experience

Plumerai

Research
